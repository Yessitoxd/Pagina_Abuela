“Desde la carpeta del repo” = la raíz del proyecto:
C:\Users\PC\Desktop\Trabajos DG Programas PC\Abuela\Pagina web

Usa PowerShell (los comandos que te doy están pensados para PowerShell en Windows). Los comandos git también funcionan en Git Bash si prefieres.
